R4D2

Requirements
============

Node.Js and MongoDB are required to run the station.


Installation
============

* Clone it from Github: `https://github.com/vvision/R4D2.git`.

##Station
* `npm install` to perform installation.
* `node server.js` to launch the API and read the serial.
